API Reference Test Coverage Changelog

Server-side world generation for Minecraft

FarLanders provides configurable, modular terrain and structure generation with tools for previewing chunks offline and a Java API for integration.

Quick actions

  • Build, test and generate Javadoc: ./gradlew clean build test javadoc
  • Preview the assembled site locally: make preview
  • Download plugin JARs from the public/download/ folder after CI runs

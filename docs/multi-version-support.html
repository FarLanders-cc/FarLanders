üåü Multi-Version Compatibility

FarLanders v1.2.0 now supports multi-version Minecraft servers through ViaVersion and ViaRewind integration. Allow players on different Minecraft versions to enjoy the Far Lands experience together!

üìã What is ViaVersion?

ViaVersion allows newer Minecraft clients to connect to older servers.

ViaRewind allows older Minecraft clients to connect to newer servers.

ViaBackwards provides additional backward compatibility for ViaVersion.

üîÑ How It Works

These plugins work by translating packets between different Minecraft protocol versions in real-time, allowing cross-version compatibility without changing your server's core version.

Step 1: Install ViaVersion Plugins

# Download the plugins (replace X.X.X with latest versions)
wget https://github.com/ViaVersion/ViaVersion/releases/download/4.9.2/ViaVersion-4.9.2.jar wget https://github.com/ViaVersion/ViaRewind/releases/download/3.0.3/ViaRewind-3.0.3.jar wget https://github.com/ViaVersion/ViaBackwards/releases/download/4.9.2/ViaBackwards-4.9.2.jar

Step 2: Plugin Installation Order

  1. ViaVersion (Core plugin - install first)
  2. ViaRewind (For older client support)
  3. ViaBackwards (For additional compatibility)
  4. FarLanders (Will auto-detect ViaVersion)

Step 3: Server Restart

Restart your server after installing all plugins. FarLanders will automatically detect and integrate with ViaVersion plugins.

üîß Troubleshooting

Common Issues

  • Plugin not detecting ViaVersion: Ensure ViaVersion loads before FarLanders
  • Material errors: Enable legacy_material_fallback in config
  • World generation issues: Check server version compatibility
  • Performance issues: Consider disabling features for very old clients

üí° Performance Tips

  • Use Paper/Purpur instead of Spigot for better ViaVersion performance
  • Limit the version range if you don't need full compatibility
  • Monitor server performance with mixed-version players
  • Consider separate worlds for different version groups if needed

Quick Start

# Build the plugin
./gradlew build
# The plugin JAR will be in:
build/libs/FarLanders-1.1.1.jar
# Install to server and restart
# Copy JAR to plugins/ folder
# Basic commands
/farlanders tp farlands # Teleport to Far Lands /farlanders generate # Generate Far Lands world /farlanders spawn setfarlands # Set spawn to Far Lands

Key Features

  • Enhanced World Generation - Beautiful terrain with Far Lands distortion
  • Progressive Terrain Complexity - Terrain becomes more unique as you explore further
  • Sky Islands - Rare floating islands with valuable resources
  • Rich Agriculture - Biome-specific farms and agricultural infrastructure
  • Enhanced Mob Spawning - Realistic animal habitats and environments
  • Advanced Spawn Management - Flexible spawn control and Far Lands integration
  • Admin Commands - Full administrative control over generation and spawn systems

Control where players spawn in your FarLands world

The FarLanders plugin now includes comprehensive spawn management functionality that allows administrators to control where new players spawn in the world.

‚ö° Commands

Permission Required: All spawn management commands require the farlanders.admin.spawn permission.

/farlanders spawn set

Sets the world spawn point to your current location.

  • Usage: /farlanders spawn set
  • Permission: farlanders.admin.spawn
  • Requirements: Must be executed by a player (not console)

/farlanders spawn reset [world]

Resets the world spawn to the default location (0, highest block, 0).

  • Usage: /farlanders spawn reset [world]
  • Permission: farlanders.admin.spawn
  • Parameters:
    • world (optional): Target world name. If not specified, uses current world for players or "world" for console.

/farlanders spawn info [world]

Displays information about the current spawn point.

  • Usage: /farlanders spawn info [world]
  • Permission: farlanders.admin.spawn
  • Parameters:
    • world (optional): Target world name. If not specified, uses current world for players or "world" for console.

üåÄ /farlanders spawn setfarlands [world]

Sets the world spawn to a Far Lands location (12.55 million blocks from origin).

  • Usage: /farlanders spawn setfarlands [world]
  • Permission: farlanders.admin.spawn
  • Parameters:
    • world (optional): Target world name. If not specified, uses current world for players or "world" for console.
  • Coordinates: Sets spawn to approximately X: 12,550,000, Z: 100
  • Safety: Automatically calculates safe Y coordinate based on terrain

Making Far Lands the Default Spawn

To make the Far Lands the default spawn point for new players:

1. Basic Setup:

/farlanders spawn setfarlands

This sets the current world's spawn to the Far Lands.

2. Specific World:

/farlanders spawn setfarlands world_name

This sets a specific world's spawn to the Far Lands.

3. Verify Setup:

/farlanders spawn info

This confirms the spawn has been set correctly.

üåç Far Lands Spawn Benefits

When players spawn in the Far Lands (at distances > 12,000,000 blocks from origin):

  • Unique Experience: Players start their adventure in the mysterious and chaotic Far Lands terrain
  • Instant Access: No need to travel millions of blocks to reach the Far Lands
  • Exploration Focus: Server experience focused on exploration and world generation features
  • Community Spawning: All new players start in the same fascinating region

üîß Technical Details

Far Lands Coordinates

  • X Coordinate: 12,550,000 (safely within Far Lands territory)
  • Z Coordinate: 100 (reasonable starting position)
  • Y Coordinate: Automatically calculated based on terrain height
  • Safety Checks: Y coordinate is bounded between 10 and 200 for safe spawning

Distance Calculation

Distance calculations for spawn management:

  • Standard Spawn: Distance calculated from world spawn (typically 0,0)
  • Far Lands Spawn: Distance calculated from Far Lands spawn point (12.55M blocks from origin)
  • World Generation: Far Lands terrain effects begin at configured threshold

Performance Considerations

  • Y-Coordinate Lookup: The system queries the highest block at the spawn location
  • Chunk Loading: Setting Far Lands spawn may require chunk generation
  • World Access: Commands work across different worlds as specified
